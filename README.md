# De Sola FC News

Portal de notícias do De Sola FC. Fique por dentro de tudo que acontece no clube.

## Tech Stack

- Next.js 16
- TypeScript
- TailwindCSS
- App Router

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Deploy

O projeto está preparado para deploy na [Vercel](https://vercel.com).
